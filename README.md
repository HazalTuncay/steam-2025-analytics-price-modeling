# steam-2025-analytics-price-modeling
Steam 2025 veri seti (240.000+ oyun) kullanılarak gerçekleştirilen bu çalışmada, oyun pazarındaki fiyatlandırma dinamikleri ve trendler analiz edilmiş; teknik ve kategorik özellikler üzerinden oyunların piyasa fiyatını tahmin eden makine öğrenmesi modelleri geliştirilmiştir.
